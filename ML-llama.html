Machine Learning con Llama 3.1

Optimización y fine-tuning de Llama 3.1 para tareas específicas, desarrollado en colaboración con Daniel Romero.

// el proyecto

¿Qué es este proyecto?

Es un proyecto de machine learning enfocado en el fine-tuning y optimización del modelo de lenguaje Llama 3.1 de Meta para adaptarlo a tareas concretas y específicas. En lugar de usar el modelo base genérico, trabajamos en entrenarlo con datos orientados a un dominio particular para mejorar su precisión y utilidad.

Fue desarrollado en colaboración con Daniel Romero Madrid, compañero del semillero de investigación en aplicaciones telemáticas de la UNIBARRANQUILLA. El trabajo conjunto permitió explorar técnicas avanzadas de ajuste de modelos LLM con recursos limitados.

// funcionalidades

¿Qué logramos?

  • Fine-tuning de Llama 3.1 para adaptar el modelo a tareas específicas de dominio
  • Optimización del proceso de entrenamiento para ejecutarse con hardware accesible
  • Evaluación y comparación del rendimiento antes y después del ajuste
  • Implementación local del modelo con Ollama para inferencia eficiente
  • Exploración de técnicas como LoRA para reducir el costo computacional del fine-tuning

// aprendizajes

¿Qué aprendí?

Este fue mi proyecto más técnico hasta la fecha. Aprendí cómo funcionan internamente los modelos de lenguaje grandes (LLMs), cómo se pueden adaptar a casos de uso específicos mediante fine-tuning, y los desafíos reales de trabajar con modelos de esta escala.

También profundicé en conceptos clave de machine learning como la preparación de datasets, la evaluación de modelos y las técnicas de optimización de parámetros. Trabajar junto a Daniel me permitió abordar el proyecto desde distintos ángulos y aprender de su experiencia en telecomunicaciones e IA.
