Calculadora de Interpolación

Herramienta web interactiva que implementa los 3 métodos numéricos de interpolación más utilizados en modelamiento matemático.

// el proyecto

¿Qué es esta calculadora?

Es una aplicación web que permite calcular valores interpolados usando los tres métodos numéricos principales: interpolación de Lagrange, Newton y lineal. Fue desarrollada como proyecto de modelamiento matemático, traduciendo conceptos del cálculo numérico en una herramienta visual e interactiva.

El usuario ingresa los puntos conocidos y el valor a interpolar, y la calculadora aplica automáticamente los métodos para entregar el resultado. Todo construido desde cero con HTML, CSS y JavaScript puro, sin librerías externas.

// funcionalidades

¿Qué puede hacer?

  • Interpolación de Lagrange — cálculo del polinomio interpolador clásico
  • Interpolación de Newton — método de diferencias divididas
  • Interpolación lineal — estimación entre dos puntos conocidos
  • Interfaz interactiva para ingresar puntos y obtener resultados en tiempo real
  • Visualización clara de los resultados para cada método

// aprendizajes

¿Qué aprendí?

Este proyecto me permitió conectar dos mundos: el cálculo numérico y el desarrollo web. Implementar los algoritmos de interpolación desde cero en JavaScript me obligó a entender a fondo la matemática detrás de cada método, no solo memorizarla.

También fue un ejercicio sólido de lógica de programación pura — sin frameworks ni librerías, solo JavaScript vanilla resolviendo problemas matemáticos reales.
