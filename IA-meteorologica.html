IA Meteorológica

Sistema de inteligencia artificial para predicción del clima construido con Rasa, desarrollado en colaboración con Daniel Romero.

// el proyecto

¿Qué es la IA Meteorológica?

Es un sistema de inteligencia artificial conversacional capaz de predecir y responder consultas sobre el clima. Fue construido con Rasa, un framework open source para crear asistentes virtuales y chatbots con procesamiento de lenguaje natural.

El proyecto fue desarrollado en conjunto con Daniel Romero Madrid, compañero del semillero de investigación en aplicaciones telemáticas de la UNIBARRANQUILLA. Ambos aportamos desde el diseño de los flujos de conversación hasta la integración con datos meteorológicos.

// funcionalidades

¿Qué puede hacer?

  • Predicción meteorológica mediante conversación natural con el usuario
  • Procesamiento de lenguaje natural con Rasa NLU para entender consultas climáticas
  • Respuestas dinámicas basadas en datos del clima en tiempo real
  • Flujos de diálogo personalizados para distintos tipos de consultas
  • Arquitectura conversacional escalable y adaptable a otros dominios

// aprendizajes

¿Qué aprendí?

Este proyecto fue mi primera experiencia real trabajando con inteligencia artificial conversacional. Aprendí a diseñar flujos de diálogo, entrenar modelos NLU y conectar un asistente virtual con fuentes de datos externas.

También fue un ejercicio valioso de trabajo en equipo: coordinar decisiones técnicas, dividir tareas y mantener coherencia en el código entre dos personas me preparó para colaboraciones más complejas.
