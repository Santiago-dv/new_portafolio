Sneaker Resell

Plataforma web full-stack para comprar y vender sneakers, con autenticación, mensajería interna y un chatbot de recomendaciones impulsado por IA.

// el proyecto

¿Qué es SneakerResell?

SneakerResell es una aplicación web construida con Flask y SQLite orientada al mercado de reventa de zapatillas. Permite a los usuarios tanto comprar como vender productos dentro de la misma plataforma, con una experiencia cercana a la de un marketplace real.

El proyecto nació como un ejercicio completo de desarrollo web: desde el diseño de la base de datos y la lógica del backend, hasta las vistas HTML/CSS del frontend. Cada funcionalidad fue pensada para resolver un problema concreto de los usuarios.

// funcionalidades

¿Qué puede hacer?

  • Registro e inicio de sesión con autenticación de usuarios
  • Publicación de productos con imágenes, precio y descripción
  • Catálogo dinámico de sneakers disponibles para compra
  • Sistema de mensajería interna entre compradores y vendedores
  • Chatbot de recomendaciones integrado, impulsado por Llama 3.2 vía Ollama

// aprendizajes

¿Qué aprendí?

Este proyecto fue mi primera experiencia construyendo una aplicación web completa de principio a fin. Aprendí a estructurar un proyecto Flask, a manejar bases de datos relacionales con SQLite, y a integrar un modelo de IA local para enriquecer la experiencia del usuario.

También fue mi primer acercamiento real a la autenticación, el manejo de sesiones y la lógica de un sistema de mensajería entre usuarios.
